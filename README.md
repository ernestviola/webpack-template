# Todo List
A todo list made as an assignment from [The Odin Project](https://www.theodinproject.com/lessons/node-path-javascript-todo-list).

## Lessons
Keep in mind the [SOLID principles](https://www.geeksforgeeks.org/system-design/solid-principle-in-programming-understand-with-real-life-examples/)
1. Single Responsibility
2. Open Closed
3. Liskov Substitution
4. Interface Segregation
5. Dependency Inversion

## Attributions

### Features
1. Create a note that is stored in a default folder
- CRUD any note
- title, description, dueDate, priority
2. Can create a note folder
- CRUD any project
3. Able to move notes between folders
4. Add local storage with the Web Storage API
